# ⏰ Relógio Digital

Este projeto é um **Relógio Digital** desenvolvido com **HTML, CSS e JavaScript**, criado com o objetivo de praticar manipulação do DOM, lógica em JavaScript e estilização com CSS.

O relógio exibe a **hora atual em tempo real**, atualizando automaticamente a cada segundo.

---

## Funcionalidades

- Exibição de horas, minutos e segundos
- Atualização automática em tempo real
- Layout simples e organizado
- Estilização personalizada
- Responsivo para diferentes tamanhos de tela

---

##  Tecnologias Utilizadas

- **HTML5** — estrutura da página  
- **CSS3** — estilização e layout  
- **JavaScript (Vanilla)** — lógica do relógio e atualização do tempo  

---

##  Objetivo do Projeto

Este projeto foi desenvolvido com fins **educacionais**, para aprimorar habilidades em:
- JavaScript básico
- Manipulação do DOM
- Organização de arquivos
- Estilização com CSS

---

## Como Executar

1. Baixe ou clone este repositório  
2. Abra o arquivo `index.html` em qualquer navegador moderno  
3. O relógio será exibido automaticamente com a hora atual
